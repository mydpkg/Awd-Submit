# Awd-Submit
CTF AWD flag自动提交工具

```
root@root: $ ./Awd-Submit_darwin_amd64 -h                                           
Usage of ./Awd-Submit_darwin_amd64:
  -t uint
    	默认每5分钟执行一次 (default 5)
  -u string
    	flag提交路径，自动附带flag，https://ctf.bugku.com?token=xxx
```
